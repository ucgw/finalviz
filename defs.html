FIELD_NAME VALUE_TYPE DESCRIPTION
id String A unique identifier for the event
magError Decimal Uncertainty of reported magnitude of the event. estimated standard error of magnitude
magNst Integer Total number of seismic stations used to calculate the magnitude for this earthquake
mag Decimal The magnitude for the event.
depthError Decimal Uncertainty of reported event (in km.)
depth Decimal Depth of event (in km.)
nst Integer Total number of seismic stations used to determine earthquake location
Date/Time Date Time Time when event occurred (in UTC)
place String Textual description of named geographic region near to the event
